# jv-oop

- Create class Machine containing methods `public abstract void doWork()` and `public abstract void stopWork()`.  
- Create classes `Truck`, `Bulldozer`, and `Excavator` that will inherit from `Machine`.
- In those classes override `doWork()`, so it will print a message that a certain machine started its work.
- Override `stopWork()` as well. It should print messages that certain machines stopped working.
- In the `MainApp` class create a `Machine` array with `Truck`, `Bulldozer`, and `Excavator` and call methods `doWork()` and `stopWork()` in a loop.

Створи клас Machine, який містить методи public abstract void doWork() та public abstract void stopWork().

Створи класи Truck, Bulldozer і Excavator, які будуть наслідувати Machine.

У цих класах перевизнач метод doWork(), щоб він виводив повідомлення про те, що відповідна машина почала працювати.

Також перевизнач метод stopWork(). Він має виводити повідомлення про те, що відповідна машина припинила роботу.

У класі MainApp створи масив Machine, який міститиме Truck, Bulldozer і Excavator, та в циклі виклич методи doWork() і stopWork().
